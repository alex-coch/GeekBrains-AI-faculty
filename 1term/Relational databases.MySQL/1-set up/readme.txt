На первом уроке, преподаватель покажет Вам как верно установить ПО. Рекомендуем до первого занятия самостоятельно ничего не устанавливать.

В материалах к уроку есть дополнительная информация:
- Как установить MySQL
- Если возникли сложности при установке, можете воспользоваться виртуальной машиной. Во вложении ссылка на видео и описание.

Если во время установки возникли сложности, обращайтесь к преподавателю / наставнику / пишите в общий чат.

Обратите внимание, что программа курса идёт под версию 5.7. Новые функции в версии 8 нам пока не понадобятся, но можете установить и последнюю версию.

Книги:
?
1) "Руководство по MySQL" Тахагхогхи С., Вильямс Хью Е. (к сожалению, издание старое. Базовый разбор SQL с нуля, написано очень понятно, читается легко)
2) "Изучаем SQL" Алан Бьюоли (издание старое. Разбор всех основных тем: создание и заполнение БД, SQL-запросы, транзакции, индексы. Есть упражнения и разбор к ним )
3) "SQL. Сборник рецептов" Энтони Молинаро (издание старое. Разбор зпрактических задач, объяснение решения и код для СУБД DB2 v.8, Oracle Database 10g, PostgreSQL 8, SQL Server 2005, MySQL 5)
4) "MySQL по максимуму". Шварц Б., Зайцев П., Ткаченко В. (2018г. Глубокий разбор базовых тем, тонкости оптимизации работы СУБД, репликация, масштабирование, облако)
5)"" Обеспечение высокой доступности систем на основе MySQL" Чарльз Белл, Мэтс Киндал, Ларс Талманн (2012г. Репликация, масштабирование, монитиринг, облачные вычислени)
6) "Введение в системы баз данных Крис Дж. Дейт (классический учебник)
7) "PostgreSQL. Основы языка SQL". Евгений Моргунов:
8) "MySQL 8 для больших данных. Эффективная обработка данных с помощью MySQL 8, Hadoop, API NoSQL и других инстументов для больших данных" Чаллавала Ш., Лакхатария Д.
9) "Семь баз данных за семь недель. Введение в современные базы данных и идеологию NoSQL" Уилсон Джим Р., Редмонд Эрик
10) "Основы технологий баз данных: учебное пособие" Б. А. Новиков, 2020г.
11) "Высоконагруженные приложения. Программирование, масштабирование, поддержка" Клеппман М., 2018
12) "Learning SQL" by Alan Beaulieu 2020 (на английском, но всё по полочкам, есть упражнения, примеры на MySQL)
