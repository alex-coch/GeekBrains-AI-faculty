- `train.csv` обучающая выборка со следующими столбцами:  
   - **id** - идентификатор временного ряда сердечного ритма
   - **time** - время в ms ( миллисекунды)
   - **x** - R-R интервал
   - **y** - признак разметки ковидной аномалии
 - `sample_submission.csv` пример формата отправки решения
 - `eval.py` - скрипт для подсчёта F1 (binary) score, принимает два позиционных аргумента:  
   - **pred_path** - путь до csv файла с предсказанием (формат файла как в `sample_submission.csv`)
   - **true_path** - путь до csv файла с истинным значенем (формат файла как в `sample_submission.csv`)
